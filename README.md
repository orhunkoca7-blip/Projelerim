# Calculator – C# Windows Forms

C# ve Windows Forms kullanılarak geliştirilmiş basit bir masaüstü hesap makinesi uygulaması.

## Özellikler
- Toplama, çıkarma, çarpma, bölme işlemleri
- Button Click event mantığı
- TextBox üzerinden ekran yönetimi
- Mouse ile etkileşimli kullanım

## Kullanılan Teknolojiler
- C#
- .NET
- Windows Forms (WinForms)
- Visual Studio

## Proje Yapısı
- Form tabanlı arayüz tasarımı
- C# ile iş mantığı
- Event-driven programlama yaklaşımı

## Amaç
Bu proje, C# WinForms temel bilgilerini göstermek ve
masaüstü uygulama geliştirme pratiği kazanmak amacıyla yapılmıştır.

- Klavye desteği eklenmesi
- Daha gelişmiş hata kontrolleri
- Arayüz iyileştirmeleri
